copy the generated code and put it into a new text file named "rageconfig.cfg" in these locations:

Legend of values
DNA very high = best quality
DNA high = good quality
default = default RAGE settings
DNA low = below default RAGE settings
DNA very low = very low settings for low spec systems
"DNA enhanced" specific on and off settings
